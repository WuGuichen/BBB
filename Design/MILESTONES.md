# 2. 各阶段详情

## M0：骨架 + 测试场

要做：
1. `Assets/Scripts/WGame/` + `WGameRuntimeCompositionRoot`
2. RuntimeHost + RuntimeCommandBuffer
3. AABB房间 + 占位生物 + 基础移动
4. 2.5D等距相机（55-60°，正交投影）
5. Sensory Sandbox调试场景：固定房间、可重置
6. Decision Trace Overlay

验收：
- 生物可移动，不穿墙
- 一键重置后状态完全一致

---

## M1：身体部件 + 基础AI + 单区Run ⚠️ 判生死

**目标**：玩家组装生物、配置AI、投放1个生态区。同一房间3次播放事件不同。

要做：
1. 身体部件系统（Core/Movement/Sensor/Weapon/Defense/Utility/Brain）×3个
2. 预算系统（体重+能量）
3. AI蓝图系统（卡片式规则：条件卡+意图卡+权重）
4. 5条基础AI规则
5. 基础状态机（Idle/Explore/Combat/Flee/Return）
6. 战争迷雾 + 视觉传感器
7. R01-Lab房间：地形 + 3-4个活物 + 环境元素
8. 单区Run结构（进入→探索→到达出口→结束）

验收（涌现版）：
- 同一房间播放3次，至少出现5种不同"主要事件"
- 3个非开发者各看一次，每人说出的故事都不一样
- 玩家能说出"我换了XX部件/改了XX规则，结果不一样了"

**不通过则停。**

---

## M2：感知 + 涌现

要做：
1. 嗅觉传感器（气味流方向，±30°误差）
2. 已揭开区域"记忆衰减"
3. 活物独立AI（巡逻者/捕食者/清道夫各有目标）
4. 内部需求系统（饥饿/疲劳/恐惧）
5. 环境规则（酸池扩散/毒气变向）
6. AI反馈四层（行为/情绪/理由推断/结果）

验收：
- 嗅觉和视觉显示方式不同
- 活物之间有自然交互
- 3个非开发者能区分"看到"和"闻到"

---

## M3：信号 + 间接指挥

要做：
1. 信号系统（移动信标/危险标记/召回信标）
2. 信号改变Utility分数，不直接控制
3. AI蓝图极端时生物不听信号
4. 信号稀缺：每次投放0-2个
5. 结果层反馈

验收：
- 谨慎核心听召回，狂暴核心不听
- 信号是稀缺的，玩家必须选择什么时候用
- 3个非开发者觉得"我在和它沟通"

---

## M4：连续Run + 撤退 ⚠️ 判生死

**目标**：3个连续生态区 + 撤退决策 + Run内部件/样本分离。

要做：
1. Run结构：3个连续生态区，难度递增
2. Run内部件（死了丢）vs 样本/数据（死了保留）
3. 撤退机制：随时可撤退，带回当前收集
4. 生态区之间过渡
5. Run内部件在生态区内拾取/掉落

验收：
- 玩家面临"继续还是撤退"的决策
- 全灭后玩家说"下次我应该早点撤"
- Run内部件死了就没了，玩家感到损失
- 样本/数据死了保留，玩家不觉得白打

**这是肉鸽核心。不通过就不是肉鸽。**

---

## M5：战后报告 + Meta进度

要做：
1. 战后报告（死亡原因/关键行为/触发规则/建议修改）
2. 回收资源（样本/数据死了也保留）
3. 实验站设施（孵化槽/部件仓/AI台）
4. 解锁系统（用样本/数据解锁新部件选项/新AI规则/新生态区）

验收：
- 玩家看完报告能说"下次我要改XX"
- 解锁后玩家说"下次Run我可以用这个"
- 失败不让人沮丧，让人想再试

---

## M6：多生态区 + 跨Run

要做：
1. 废弃温室（基础生存）
2. 酸性洞穴（抗性测试）
3. 机械废墟（潜行测试）
4. 跨Run持久化（尸体/环境变化保留）
5. 生态位多样化（草食者/捕食者/清道夫/机械巡逻者）

验收：
- 同一生物在不同生态区表现不同
- 玩家会说"这个生物适合温室但不适合洞穴"
- 跨Run后地图有变化

---

## M7：队伍 + 多生物

要做：
1. 多生物投放（2-3只）
2. 队伍AI（保护/协作/呼叫）
3. 半暂停机制（0.25x慢放）
4. 注意力分配UI

验收：
- 玩家能同时关注2-3只生物
- 队伍协作有策略感
- 如果焦虑就回到单生物

---

## M8：完整Roguelite

要做：
1. 5个连续生态区
2. Eco Director（根据世界状态生成事件）
3. 完整meta进度（科技树）
4. Run总结

验收：
- 每次Run不同
- 玩家完成Run后想再试
- meta进度有意义

---

# 3. MVP最小闭环

```
玩家组装1只生物（3个部件 + 5条AI规则）
↓
投放到1个生态区（废弃温室）
↓
生物自主行动（按AI规则）
↓
玩家用1-2个信号干预
↓
到达出口或死亡
↓
Run结束：样本保留，Run内部件丢失
↓
战后报告（死亡原因/建议）
↓
玩家改设计/用数据解锁新选项
↓
再试
```

**这个闭环在M4完成。** M4 = MVP。
