# MILESTONES — 制作计划

> 版本 0.8 | 2026-05-27

---

# 1. 阶段总览

**M1-M2 是判生死节点。** 如果一只生物在迷雾房间里走 60 秒，找东西、犹豫、被你喊一句话改变了方向 — 如果这一分钟让 playtest 玩家睁大眼睛，项目就成立。

| 阶段 | 目标 | 核心验证 |
|------|------|----------|
| **M0** | 骨架 | RuntimeHost + 占位生物 + AABB 房间 |
| **M1** | 感知与迷雾 | 一只生物的传感器揭开未知，玩家通过它的眼睛看世界 |
| **M2** | 性格与犹豫 | AI 模块改变行为，矛盾叠加产生犹豫，性格可读 |
| **M3** | 第二感知模态 | 嗅觉 = 气流方向，视觉差异验证 |
| **M4** | 干预动词 | 4 个喊话动词，性格敏感（勇气不听安抚） |
| **M5** | 意外与遭遇 | 第一个敌人，定位是"探险事件"不是"关卡 boss" |
| **M6** | 第二只生物 + 半暂停 | 验证注意力分配是否成立 |
| **M7** | 信任度 | 你帮过它它更听话，你坑过它它不理你 |
| **M8** | 零件 + Run | 探索获得零件、背包、核心零件不丢、Run 结构 |

---

# 2. 各阶段详情

## M0：骨架

要做：
1. `Assets/Scripts/VVV/` + `VVVRuntimeCompositionRoot`
2. `RuntimeHost` + `RuntimeCommandBuffer`
3. AABB 房间 + 占位生物 + 基础移动

验收：生物可移动，不穿墙，Replay hash 一致。

---

## M1：感知与迷雾 ⚠️ 判生死

**目标**：一只生物的传感器揭开未知，玩家通过它的眼睛看世界。

要做：
1. 战争迷雾（Unity 视觉层，生物走到哪里揭开哪里）
2. 视觉传感器（CombatPhysicsWorld Sphere 查询 → 揭开圆形范围）
3. 地图内容：小而密的房间，5-10 个信息事件点（资源/宝箱/分叉/危险区）
4. 生物自主移动到目标 + 探索行为（走到分叉时选路）
5. 信息密度保证：每 5-10 秒至少一个新信息（揭开新区域/发现东西/停下）

验收：
- 玩家能通过生物的"眼睛"看到未知环境被逐步揭开
- 信息密度：60 秒内至少 6 个信息事件
- **主观 Playtest**：3 个非开发者看 60 秒后说"我想知道前面有什么"

**如果这个 playtest 不通过，停下来重新评估。**

---

## M2：性格与犹豫 ⚠️ 判生死

**目标**：AI 模块改变行为，矛盾叠加产生犹豫，性格一句话可读。

要做：
1. AI 模块（勇气核心 / 谨慎核心 / 贪欲核心）
2. 模块矛盾叠加（勇气+谨慎 = 高血量冲低血量逃）
3. 犹豫表现（"？" + 减速 + Brain View 浮现）
4. 性格高亮（做了什么 + 没做什么）
5. 短期记忆 v0（被攻击过的方向 → 下次更警惕）
6. 干预动词 v0（"警告" — 一个动词就够，先验证干预是否成立）

验收：
- 装勇气核心：看到资源直接去拿，不犹豫
- 装谨慎核心：看到资源先停一下再决定
- 装勇气+谨慎：血量高时冲，血量低时跑，中间犹豫
- 犹豫时"？"可见
- **主观 Playtest**：3 个非开发者能说出"这只生物的性格是 XX"和"它刚才犹豫了因为 XX"
- 60 秒内至少 2 个性格高亮事件

**如果 playtest 玩家觉得"它就是随机在走"，停下来。性格必须可读。**

---

## M3：第二感知模态

**目标**：嗅觉 = 气流方向，两种感知的视觉差异清晰。

要做：
1. 嗅觉传感器（检测范围内标记实体 → 显示气味流方向）
2. 气味流视觉（红色 = 威胁方向，绿色 = 资源方向，无精确位置）
3. 嗅觉产生 AI fact（`threat.smell` confidence 0.5-0.7）
4. 两种感知同时存在的视觉差异

验收：
- 嗅觉和视觉的显示方式明显不同
- 嗅觉检测到威胁时，生物根据性格做出不同反应（勇气调查/谨慎绕路）
- **主观 Playtest**：3 个非开发者能区分"看到敌人"和"闻到敌人"

---

## M4：干预动词

**目标**：4 个喊话动词，性格敏感。

要做：
1. 警告（AvoidThreat +30）— 勇气核心可能忽略
2. 建议（MoveToTarget +30）— 谨慎核心可能不听
3. 安抚（Caution +20, Aggression -20）— 勇气核心听不进
4. 鼓舞（Aggression +20, Caution -20）— 谨慎核心听不进
5. 信号道具（探索中获得，一次性）
6. 干预资源 UI

验收：
- 警告让谨慎核心绕路，但勇气核心可能忽略
- 安抚让犹豫的生物冷静下来，但纯勇气型不受影响
- 信号是稀缺的（一局 0-3 个）
- **主观 Playtest**：3 个非开发者觉得"我在和它对话"

---

## M5：意外与遭遇

**目标**：第一个敌人，定位是"探险事件"。

要做：
1. 敌人（巡逻，发现生物后追击）
2. 接触伤害（3-5 秒战斗，0.3 秒前摇 + 音效 + 伤害数字）
3. 生物根据性格决定打/跑/犹豫
4. 战斗后继续探索（战斗不是终点，是过程中的一个事件）

验收：
- 战斗在 5 秒内结束，结果可读
- 勇气核心主动打，谨慎核心绕开
- 勇气+谨慎叠加时：高血量冲，低血量逃
- 战斗不是高潮 — **主观 Playtest**：3 个非开发者觉得"战斗是探险的一部分，不是重点"

---

## M6：第二只生物 + 半暂停

**目标**：验证注意力分配是否成立。

要做：
1. 第二只生物投放（各走各路）
2. 半暂停机制（手动 0.25x 慢放）
3. 注意力分配 UI（两只的传感器信息同时显示）

验收：
- 两只生物各走各路时，玩家能同时关注
- **主观 Playtest**：3 个非开发者 — 如果觉得"焦虑"就回到单生物路线；如果觉得"有趣"就继续

---

## M7：信任度

**目标**：你帮过它它更听话，你坑过它它不理你。

要做：
1. TrustLevel 属性（初始 50）
2. 信号帮助了生物 → Trust +20
3. 信号坑了生物 → Trust -30
4. Trust 低时信号效果减弱
5. Trust 高时犹豫时刻更倾向听你的

验收：
- 救过它之后，下次犹豫时更听你的信号
- 坑过它之后，下次不太听了
- **主观 Playtest**：3 个非开发者能说出"它现在不太听我的了，因为上次我让它冲结果害死了它"

---

## M8：零件 + Run

**目标**：探索获得零件，组装生物，Run 结构。

要做：
1. 零件掉落（宝箱/敌人掉落，随机）
2. 背包（8 格，满时丢弃）
3. 投放前组装（选身体部件 + AI 模块）
4. 核心零件死亡不丢
5. Run 结构（3 个连续房间，难度递增）
6. Run 总结

验收：
- 每局零件不同 → 生物性格不同 → 体验不同
- 核心零件不丢 → 失败有代价但不挫败
- **主观 Playtest**：3 个非开发者完成 Run 后想再试

# 3. 最小竖切

| 类别 | 内容 |
|------|------|
| **感知** | 视觉（圆形揭开）+ 嗅觉（气流方向） |
| **生物** | 1 只，HP/MoveSpeed/Attack |
| **AI 模块** | 勇气 / 谨慎 / 贪欲 |
| **干预** | 警告 / 建议 / 安抚 / 鼓舞 |
| **零件** | 快腿/厚甲/毒牙/嗅觉腔/勇气/谨慎/贪欲 |
| **地图** | 小而密的迷雾房间，5-10 个信息事件 |
| **UI** | 背包 / 信号 / 犹豫提示 / 性格高亮 |

# 4. Playtest 节点

| 阶段 | 内容 | 不通过则 |
|------|------|----------|
| **M1** | "我想知道前面有什么" | 停下来重新评估 |
| **M2** | "这只生物的性格是 XX" | 停下来，性格必须可读 |
| **M3** | 能区分"看到"和"闻到" | 感知视觉语言要改 |
| **M4** | "我在和它对话" | 干预动词要改 |
| **M5** | "战斗是探险的一部分" | 战斗太抢眼要降级 |
| **M6** | 焦虑还是有趣？ | 焦虑就回到单生物 |
| **M7** | "它不太听我的了" | 信任度要更明显 |
| **M8** | 想再试一次 | Run 结构要改 |

---

# 5. 风险

| 风险 | 应对 |
|------|------|
| 探索节奏太慢 | 地图小而密，信息事件每 5-10 秒 |
| 看不懂生物在想什么 | 感知可视化 + 性格高亮 + M1/M2 Playtest |
| 战斗抢走注意力 | 短而突然（3-5秒），定位是事件 |
| 多生物焦虑 | M6 验证，不成立就回到单生物 |
| 性格可预测就无聊 | 短期记忆 + 矛盾叠加 + 信任度 |
| 感知可视化成本高 | MVP 只做视觉+嗅觉，听觉 Post-MVP |

---

# 6. 暂缓

| 功能 | 原因 |
|------|------|
| 听觉模态 | 先验证视觉+嗅觉 |
| 信任度 | M7 才做 |
| 多生物小队 | M6 验证后决定 |
| 分支星图 | 先做连续房间 |
| 零件升级 | 先验证核心循环 |
| 生态/Mod/多人 | 不在 MVP |
