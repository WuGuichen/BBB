# 2. 各阶段详情

## M0：骨架 + 测试场

要做：
1. `Assets/Scripts/WGame/` + `WGameRuntimeCompositionRoot`
2. RuntimeHost + RuntimeCommandBuffer
3. AABB房间 + 占位生物 + 基础移动
4. 2.5D等距相机（55-60°，正交投影）
5. Sensory Sandbox调试场景：固定房间、可重置、无限信号
6. Decision Trace Overlay

验收：
- 生物可移动，不穿墙
- 一键重置后状态完全一致
- Decision Trace显示当前AI决策

---

## M1：身体部件 + 基础AI ⚠️ 判生死

**目标**：玩家能组装生物、配置简单AI、投放观察。同一房间3次播放事件不同。

要做：
1. 身体部件系统（Core/Movement/Sensor/Weapon/Defense/Utility/Brain）×3个
2. 预算系统（体重+能量）
3. AI蓝图系统（卡片式规则：条件卡+意图卡+权重）
4. 5条基础AI规则（看到敌人/血量低/看到食物/看到资源/看到任务物品）
5. 基础状态机（Idle/Explore/Combat/Flee/Return）
6. 战争迷雾 + 视觉传感器
7. R01-Lab房间：地形 + 3-4个活物 + 环境元素

验收（涌现版）：
- 同一房间播放3次，至少出现5种不同"主要事件"
- 没有任何2次有80%相同的事件序列
- 3个非开发者各看一次，每人说出的故事都不一样
- 玩家能说出"我换了XX部件/改了XX规则，结果不一样了"

**诊断问题**：
- "你觉得它刚才在做什么？"
- "你再看一次，会发生什么不同的事？"
- "你想改什么让它做得更好？"

**不通过则停。** 核心循环是否成立。

---

## M2：感知 + 涌现

**目标**：嗅觉/视觉差异 + 活物独立运作 + 环境独立规则。

要做：
1. 嗅觉传感器（气味流方向，±30°误差）
2. 已揭开区域"记忆衰减"（颜色冷掉60%，动态实体不更新位置）
3. 活物独立AI（巡逻者/捕食者/清道夫各有目标）
4. 内部需求系统（饥饿/疲劳/恐惧）
5. 环境规则（酸池扩散/毒气变向）
6. AI反馈四层（行为/情绪/理由推断/结果）

验收：
- 嗅觉和视觉的显示方式明显不同
- 活物之间有自然交互（清道夫吃尸体、巡逻者被酸池逼走）
- 3个非开发者能说出"看到"和"闻到"的区别

---

## M3：信号 + 间接指挥

**目标**：有限信号 + 性格抵抗 + 玩家觉得"在和它沟通"。

要做：
1. 信号系统（移动信标/危险标记/召回信标）×3个
2. 信号改变Utility分数，不直接控制
3. AI蓝图极端时生物不听信号
4. 信号稀缺：每次投放0-2个
5. 结果层反馈："它点了点头" / 头顶✗

验收：
- 谨慎核心听召回，狂暴核心不听
- 信号是稀缺的，玩家必须选择什么时候用
- 3个非开发者觉得"我在和它沟通"

---

## M4：任务 + 战后报告

**目标**：回收任务 + 死亡原因分析 + "我知道怎么改了"。

要做：
1. 任务系统（回收任务：找到资源+带回出口）
2. 战后报告（死亡原因/关键行为/触发规则/建议修改）
3. 回收资源（死亡也给数据）
4. 任务失败后"我知道怎么改了"

验收：
- 玩家看完报告能说出"下次我要改XX"
- 失败不让人沮丧，让人想再试
- 死亡有代价但有信息价值

---

## M5：实验站 + 成长

**目标**：部件仓库 + AI蓝图台 + 科技树 + 生物质/合金/数据。

要做：
1. 实验站设施（孵化槽/部件仓/AI台/分析屏/任务终端）
2. 成长资源（生物质/合金/数据）
3. 科技树（Body/Sensor/Brain/Habitat/Field Tools）
4. 新部件解锁
5. 新AI规则解锁

验收：
- 玩家有"我解锁了新东西"的感觉
- 每次任务后有明确的成长反馈
- 科技树让人想"下一步解锁什么"

---

## M6：多生态区

**目标**：3个生态区 + 不同测试 + 跨Run持久化。

要做：
1. 废弃温室（基础生存）
2. 酸性洞穴（抗性测试）
3. 机械废墟（潜行测试）
4. 跨Run持久化（尸体/环境变化保留）
5. 生态位多样化（草食者/捕食者/清道夫/机械巡逻者）

验收：
- 同一生物在不同生态区表现不同
- 玩家会说"这个生物适合温室但不适合洞穴"
- 跨Run后地图有明显变化

---

## M7：队伍 + 多生物

**目标**：2-3只生物同时投放 + 队伍AI + 注意力分配。

要做：
1. 多生物投放
2. 队伍AI（保护/协作/呼叫）
3. 半暂停机制（0.25x慢放）
4. 注意力分配UI

验收：
- 玩家能同时关注2-3只生物
- 队伍协作有策略感
- 如果焦虑就回到单生物

---

## M8：完整Run

**目标**：连续任务 + Eco Director + Roguelite结构。

要做：
1. Run结构（3-5个连续任务，难度递增）
2. Eco Director（根据世界状态生成事件）
3. Run总结
4. 核心零件死亡不丢

验收：
- 每次Run不同
- 玩家完成Run后想再试
- Eco Director产生有意义的事件

# 3. MVP最小闭环

```
玩家设计1只生物（3个部件 + 5条AI规则）
↓
投放到1个生态区（废弃温室）
↓
生物自主行动（按AI规则）
↓
玩家用1-2个信号干预
↓
成功/失败
↓
战后报告（死亡原因/建议）
↓
玩家改设计
↓
再试
```

**这个闭环在M4完成。** M4 = MVP。

# 5. 美术方案

低模风格，部件化可重复利用。

占位阶段：圆柱体+颜色区分。不需要美术也能做出可读状态。

所有生物共用基础占位体，只用颜色和大小区分。
