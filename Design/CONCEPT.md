# CONCEPT — Creature Roguelike

> 版本 0.6 | 2026-05-27

## 一句话定位

一款 Roguelike 即时策略游戏。你用探索获得的零件组装有性格的生物，让它们在危险环境中自动执行任务，你专注于策略和关键时刻的干预。

> **用手里的牌，打出最好的局。**

## 核心公式

```
稀缺零件 + 环境压力 → 倒逼策略设计 → 自动执行验证 → 成就感
```

## 核心循环

```
进入一个区域（有关卡目标和敌人配置）
↓
探索中获得随机零件（身体部件 / AI 模块 / 信号道具）
↓
用手里的零件快速组装生物（策略决策）
↓
投放生物执行任务（自动执行，玩家观察）
↓
关键时刻用有限信号干预（稀缺的干预资源）
↓
成功 → 进入更难的区域，保留零件
失败 → 失去部分零件，回到起点
```

## 这是什么游戏

像 Slay the Spire，但不是回合制卡牌，而是实时观察 + 生物组装。

| StS | 本项目 |
|-----|--------|
| 随机发牌 | 探索获得零件 |
| 用手里的牌搭组合 | 用手里的零件搭生物 |
| 打牌出招 | 生物自动执行 |
| 看伤害数字 | 看生物行动过程 |
| 每局牌库不同 | 每局零件不同 |

**关键相同点：你不能选你想要的牌，你只能用手里的牌打出最好的局。**

## 为什么 AI 自动执行不是缺点

不是"看 AI 表演"。而是：

- **我没法同时操控** — 操作占满注意力，没空想策略
- **AI 操控让我专注策略** — 生物自己走、自己打、自己拾取
- **我的注意力全部放在**：下一步拿什么零件、怎么组合、在哪发信号

这就是"即时策略卡牌"。编排的质量决定胜负，不是操作的速度。

## 四大设计支柱

| 支柱 | 一句话 |
|------|--------|
| **零件稀缺** | 你不能选你想要的部件，你只能用手里的牌打出最好的局 |
| **压力倒逼策略** | 时间、敌人、资源有限，逼你在不完美条件下快速判断 |
| **性格可预测** | 生物不是随机行为，它有鲜明性格 — 你能预测它、但不能完全控制它 |
| **失败驱动学习** | 失去零件不是惩罚，是信息 — 告诉你"这个方案在这种环境下不行" |

## 核心风险

| 风险 | 解决 |
|------|------|
| 间接控制不如手操 | 手操根本做不了 — 零件随机，策略才是核心 |
| 策略有最优解 | 每局零件不同，没有固定最优解，只有当前手牌的最优打法 |
| 看 AI 走路无聊 | 任务短（30-60秒），关键时刻有张力，性格鲜明到一句话能描述 |
| 认知负担高 | 零件少、性格鲜明、结果一句话可解释 |

## 竞品对比

| 游戏 | 相似点 | 不同点 |
|------|--------|--------|
| **Slay the Spire** | 随机构筑 + 策略验证 + Roguelike | 回合制卡牌 vs 实时生物行动 |
| **FTL** | 随机资源 + 压力决策 + 实时干预 | 太空飞船 vs 生物组装 |
| **Into the Breach** | 间接控制 + 策略优先 + 信息透明 | 棋盘格 vs 自主行动 |
| **Monster Train** | 随机构筑 + 多层战斗 + 自动执行 | 纯卡牌 vs 生物系统 |
| **The Bibites** | 自主生物 + 生态 + 自然选择 | 无玩家干预 vs 玩家是策略核心 |

## 玩家身份

拾荒者 / 探险者 — 在废弃的实验设施和危险生态区中探索，收集零件，组装造物，完成任务，深入更危险的区域。
