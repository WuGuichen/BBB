# CONCEPT — AI Creature Sandbox

> 版本 0.5 | 2026-05-27

## 一句话定位

玩家不是直接操控角色，而是**设计生物的身体和 AI 行为逻辑**，投放到危险环境中观察它们自主行动，通过分析失败、回收资源来迭代改造。

> **设计身体，编排行为，投放生态，观察结果，回收改造。**

## 核心循环

```
提出行为假设
↓
调身体与行为旋钮
↓
快速测试
↓
投放歧义地图
↓
用有限信号干预
↓
捕捉涌现时刻
↓
查看 Attempt Diff
↓
微调并复测
```

**玩家不是看 AI 表演，玩家是在做行为实验。**

## 核心卖点

> **调教一个半自主 AI 生物，理解它为什么这么做，从失败中获得可执行的知识。**

这不是传统 RPG（控制角色打怪升级），也不是纯观察模拟（看 AI 自己跑）。它是：

```
行为实验器
+
半自主生物调教
+
可解释的涌现
+
失败驱动的学习
```

## 四大设计支柱

| 支柱 | 一句话 |
|------|--------|
| **身体决定能力** | 部件不是美术，是玩法数据。腿决定移动方式，眼决定感知范围，甲壳决定防御。 |
| **AI 决定性格** | 同一身体，不同旋钮配置有完全不同表现。胆小拾荒者 vs 狂暴猎手。 |
| **世界测试设计** | 每个环境是一张考试卷。酸沼测试抗性，暗洞测试听觉，废墟测试潜行。 |
| **失败必须有价值** | 死亡不是惩罚，是信息。因果报告、Attempt Diff、涌现高亮。 |

## 核心风险：涌现 vs 噪声

这个品类最大的陷阱：**当 AI 表现得"还行"时玩家觉得无聊；当 AI 表现得"出乎意料"时玩家不知道是自己设计巧妙还是系统耦合的偶然。**

Rain World 成立是因为每次"惊喜"都能被玩家事后归因。Spore 失败是因为改了身体却感受不到行为差异。

本项目的解法：
- **Brain View 让 AI 决策可归因**（自然语言解释，不是数字）
- **Attempt Diff 让修改可感知**（"和上次相比，你加的 smell_sensor 导致绕路了"）
- **涌现高亮让惊喜被看见**（"它记住了危险，主动绕路了"）

## 竞品对比

| 游戏 | 相似点 | 不同点 |
|------|--------|--------|
| **Ecosystem** | DNA 生成身体 + 神经系统 + 战斗能力 + 进化 | 水下生态为主；无玩家 AI 蓝图编辑 |
| **Rain World** | 程序化动画 + 捕食者/猎物生态 | 2D；玩家不创造生物/AI |
| **Niche** | 基因适应 + 生态位 + biome 筛选 | 回合制；无程序化运动 |
| **Spore** | 身体编辑 + 部件决定能力 | 生态和 AI 深度不足；战斗偏轻 |
| **Screeps** | 玩家写行为逻辑让单位自主行动 | 纯程序员游戏；无身体构筑 |
| **The Bibites** | 数字生物 + 神经网络 + 自然选择 + 生态 | 无人工设计；玩家只能观察不能干预 |
| **The Sapling** | 身体编辑 + 生态模拟 + 物种进化 | 偏进化模拟，非任务制构筑 |
| **Creatures (1996)** | 数字生命 + 神经网络 + 繁殖 + 玩家教导 | 无任务/战斗；偏宠物养成 |

## 核心差异化

> **玩家设计身体 + 玩家调教 AI 行为 + 生物自主行动 + 可解释涌现 + 确定性回放可分析**

与 The Bibites 的关键区别：The Bibites 是纯观察自然选择，玩家不能干预设计。本项目玩家是**主动设计者和调教者**，不是观察者。

## 玩家身份

边境实验站管理员 — 用半生物半机械的造物探索危险生态区，回收资源，解锁身体部件和 AI 模块。

## 首个可玩切片：Behavior Lab Slice

完整愿景很大，但第一版只验证：

- **玩家能不能通过旋钮调教 AI 行为？**
- **AI 行为差异是否可感知、可解释？**
- **失败是否能给玩家可执行的知识？**
- **"涌现时刻"是否能产生爽感？**

具体包含：
- 1 张歧义地图（三路选择）
- 5 个行为旋钮（Aggression/Caution/Curiosity/Greed/HomeBias）
- 2 个信号（Avoid/Recall，带 cooldown）
- 3 个 AI Memory（lastThreat/lastResource/avoidZone）
- Quick Test（30 秒快速验证）
- Attempt Diff（对比两次尝试）
- 涌现高亮（AI 聪明行为被系统喊出来）
- Retry Same Seed + Variant Retry

**M3 的主观验收**：让 3 个非开发者看完一次任务，能复述"这只生物刚才为什么这么做"。

## 目标玩家

这是一个**非常硬核的细分赛道**。核心玩家画像：喜欢系统型游戏、享受设计和调试过程、对"AI 自主行为"有好奇心的玩家。介于 Screeps（纯程序员）和 Spore（纯休闲）之间。
