# CONCEPT — Companion Exploration Roguelite

> 版本 0.9 | 2026-05-27

## 一句话定位

你带着你创造的生命去探险未知世界。你通过它的眼睛看世界，它根据性格做选择，你在关键时刻帮它一把。

> **通过他者的感知，理解未知。**

## 核心动词

```
释放 → 观察 → 理解 → 干预 → 接受结果
```

不是"组装"，不是"战斗"，不是"摆位置"。
是**陪伴一只你创造的生命，在未知中共同经历。**

## 核心公式

```
稀缺零件 + 未知环境 + 生物性格 → 涌现行为 → 你理解它 → 你们共同生存
```

## 核心循环

```
获得零件，组装一只生物（定义它的能力和性格）
↓
释放它进入未知环境（战争迷雾，它的传感器是唯一信息来源）
↓
它根据性格自主行动（探索/犹豫/战斗/逃跑/拾取）
↓
你通过它的感知"看到"世界（视觉/嗅觉/听觉，各有不同视觉语言）
↓
关键节点你喊话干预（建议/警告/安抚/鼓舞，它可能不听）
↓
结果：它活下来了 / 它死了 / 它带回了东西
↓
你理解了它——"它就是这样，遇到危险会冲上去"
↓
用更好的零件强化它，进入更深的未知
```

## 这是什么游戏

最接近的参照：

| 参考 | 相似点 |
|------|--------|
| **Pikmin** | 分配角色 → 自主行动 → 信号干预 → 注意力分配 → 死亡有代价 |
| **Lemmings** | 角色按性格执行 → 你在关键节点改变它 |
| **Darkest Dungeon** | 角色有可预测但不可控的心理 → 你看着他们挣扎 |
| **Lifeline** | 你只能通过有限信息"听到"状态，做建议 |

**这些游戏的玩家不会说"我赢了"，他们说"我让我的小家伙活下来了"。**

## 为什么 AI 自动执行不是缺点

不是"看 AI 表演"。而是：

- **信息不对称** — 你不知道前方有什么，生物的传感器是唯一来源
- **性格冲突** — 多个 AI 模块叠加产生真实犹豫
- **信任关系** — 你喊对了它更信任你，喊错了它不听了
- **注意力分配** — 多只生物各走各路，你只能关注一个

**自走棋的棋子没有记忆没有关系。你的生物有。**

## 四大设计支柱

| 支柱 | 一句话 |
|------|--------|
| **感知即探索** | 未知环境 + 生物传感器 = 持续接收新信息 |
| **性格即深度** | AI 模块矛盾叠加，产生可预测但不完全可控的行为 |
| **稀缺即策略** | 零件是捡的不是选的，背包有限，信号是道具 |
| **信任即关系** | 表现先于系统：MVP 阶段就有"它听了 / 没听 / 迟疑"的反馈；信任度的完整系统在 Post-MVP |

## 核心体验时刻

**发现**：嗅觉腔检测到前方有东西 — 但不知道是什么 — 它停下来了

**意外**：贪欲生物在捡资源 — 敌人从侧面出现 — 它犹豫了

**理解**：换了勇气核心 — 同样遇到敌人 — 这次它没犹豫直接冲了 — 你终于懂了"上次为什么会输"

**陪伴**：它残血了 — 你喊它回来 — 它犹豫了一下 — 然后回来了 — 屏幕底部："它点了点头"

## 生物是角色，不是单位

它有名字、有伤痕、会记得危险方向、会在归来时短暂靠近你。它死了你会难过。

> **玩家爱上的不是系统，是那只东西。**

## 核心风险

| 风险 | 解决 |
|------|------|
| 探索节奏太慢 | 地图小而密，每 5-10 秒一个信息事件 |
| 看不懂生物在想什么 | 多层感知视觉语言 + AI 四层反馈（行为/情绪/理由/结果） |
| 战斗抢走注意力 | 战斗短而突然（3-5 秒），定位是"事件"不是"高潮" |
| 陪伴幻想立不起来 | Trust v0 表现层 M3 就要可见，不等系统层 |
| 性格可预测就无聊 | 短期记忆 + 环境敏感 + 矛盾叠加 |

## 玩家身份

拾荒者 / 陪伴者 — 不是指挥官，是同伴。你和你创造的生命一起探险。

> MVP 阶段陪伴**一只**生物。多生物（注意力分配玩法）是另一种核心动词，留给 Post-MVP。

## 首个可玩切片

**M1-M2 是判生死节点。** 一只生物在迷雾房间里走 60 秒 — 找东西、犹豫、被你喊一句话改变了方向 — 如果这一分钟让 playtest 玩家睁大眼睛，项目就成立。
