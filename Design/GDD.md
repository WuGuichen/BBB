# 1. 玩家身份与世界观

玩家是**边境实验站管理员**（方案4"失控实验室管理员" + 方案1"荒野生物工程师"的混合）。

- 通过回收基因、骨骼、机械部件，创造适应环境的半生物半机械造物
- 将造物投放到危险生态区执行任务
- 回收资源，解锁新部件和 AI 模块
- 弱化人类角色，视觉重点放在怪物、机械生物、低模生态、实验设施

---

# 2. 核心循环

## 主循环

```
1. 接受任务 / 选择生态区
2. 分析环境威胁
3. 设计或选择生物（身体部件 + AI 蓝图）
4. 投放生物
5. 间接干预 / 观察行动
6. 回收资源和数据
7. 分析失败或成功原因
8. 解锁新部件、新行为、新环境
9. 继续改造
```

## 时间尺度

| 尺度 | 内容 |
|------|------|
| **30 秒微循环** | 生物发现了什么？为什么这么做？安全吗？要不要发信号？AI 是否误判？ |
| **5 分钟循环** | 投放 1~3 个生物，完成 1 个目标，发生 1~2 次意外，带回资源或尸体 |
| **30 分钟循环** | 完成一组生态区挑战，解锁新部件/AI 规则，扩建实验站 |

# 3. 身体系统

## 3.1 八类部件

| 类型 | 决定什么 | 示例 |
|------|----------|------|
| **Core 核心** | 基础形态、能量容量 | 小型核心（轻/脆）、重型核心（厚/慢）、实验核心（多AI槽/不稳定） |
| **Locomotion 移动** | 移动方式 | 双足（均衡）、四足（稳定追猎）、六足（地形适应）、履带（稳定噪音大）、翅膀（跨地形/高能耗）、触手（爬墙/复杂） |
| **Sensor 感知** | AI 能知道什么 | 普通眼、夜视眼、热感器、大耳（听觉）、嗅觉腔（追踪）、群体天线（共享情报） |
| **Weapon 攻击** | 攻击方式 | 爪、牙、角、毒刺、喷酸腺、电击器、远程骨刺、机械锯 |
| **Defense 防御** | 生存能力 | 甲壳、鳞片、脂肪层、再生组织、反伤尖刺、酸抗脚垫、伪装皮肤 |
| **Utility 功能** | 特殊能力 | 携带囊、挖掘爪、采集口器、开门工具、能量电池、隐身膜 |
| **Brain 脑模块** | AI 复杂度 | 原始脑（少量规则）、猎手脑（攻击加成）、群体脑（协作规则）、战术脑（掩体/包抄）、异常脑（强但失控） |
| **Social 社交** | 群体行为 | 信息素腺体、吼叫器官、信号天线、群体同步神经、恐吓面具 |

## 3.2 部件属性

每个部件带标签：重量、能耗、噪音、耐久、维修难度、生物质消耗、金属消耗、恐惧影响、攻击倾向影响、适配地形、兼容部件、AI 需求。

## 3.3 预算约束

防止堆满最强部件：体重预算、能量预算、脑容量预算、突变稳定度、部件槽位、维护成本。

> 重甲 + 大脑 + 六足 + 远程武器 → 速度慢、饥饿快、维护贵、容易过热。构筑有取舍。

# 4. AI 蓝图系统

## 4.1 卡片式规则

玩家看到的不是代码，而是规则卡：

```
当【看到敌人】
并且【敌人距离 < 5m】
并且【自己血量 > 50%】
则【近战攻击倾向 +60】
```

玩家组合：条件卡、比较卡、意图卡、权重卡、冷却卡、状态卡。

## 4.2 基础状态

MVP 用 9 个状态：Idle / Explore / Forage / Combat / Flee / Recover / Return / Protect / Nest

## 4.3 意图 Intent

ActionRunner 执行的意图：MoveToTarget / AttackTarget / FleeFromThreat / EatFood / InvestigateStimulus / ReturnToNest / ProtectAlly / CarryItem / UseAbility / Hide / CallForHelp / Rest

## 4.4 AI Profile（预制性格）

MVP 不开放自由编辑，给 5 个预制 Profile：

| Profile | 行为特征 |
|---------|----------|
| **Scavenger 拾荒者** | 避战，优先吃尸体和捡资源，装满回家 |
| **Hunter 猎手** | 主动追击，受伤后更凶，很少撤退 |
| **Coward 胆小鬼** | 高逃跑优先级，避免强敌，适合侦察 |
| **Guardian 护卫者** | 保护队友，吸引仇恨，中等攻击 |
| **TaskFirst 任务优先** | 忽略干扰，直奔目标，完成即撤 |

## 4.5 Utility 打分示例

**拾荒者**：

```
FleeScore   = ThreatNearby*60 + SelfHealthLow*80 + Alone*30 - AllyNearby*25
Scavenge    = CorpseNearby*70 + Hunger*40 - ThreatNearby*50
ReturnScore = InventoryFull*80 + SelfHealthLow*40 + NightTime*20
```

**狂暴猎手**：

```
AttackScore = EnemyVisible*70 + EnemyWounded*40 + SelfAnger*50 - SelfHealthLow*20
FleeScore   = SelfHealthCritical*40 - SelfAnger*50
```

# 5. 玩家介入方式

## 5.1 任务前设计（主要玩法）

在实验站配置：身体部件、感知器官、行为蓝图、性格参数、群体规则、撤退策略、目标优先级。

## 5.2 任务中间接指挥

不直接控制，只用有限信号改变 AI Utility 分数：

| 信号 | 效果 | 限制 |
|------|------|------|
| 移动信标 | 提高前往区域倾向 | — |
| 召回信标 | 提高返回倾向 | 狂暴型可能忽略 |
| 猎物标记 | 提高攻击某目标倾向 | 胆小型可能反而逃跑 |
| 安抚脉冲 | 降低恐惧/愤怒 | — |
| 刺激药剂 | 提高速度/攻击，增加失控风险 | 短效 |
| 诱饵 | 制造虚假感知 | 有限次数 |

**关键设计**：如果 AI 设计太极端，生物可能不听信号。狂暴型忽略召回，饥饿型忽略探索信标。这让生物更像"活的东西"。

## 5.3 任务后分析

完整战后报告：死亡原因、关键决策、触发过的规则、未执行动作原因、资源收益、AI 改进建议、身体弱点、环境适应性评分。

> 这是玩家真正成长的地方。参见 §10 反馈系统。

# 6. 战斗系统

## 6.1 五种战斗模式

| 模式 | 核心 | 适合场景 |
|------|------|----------|
| **捕食战斗** | 发现→追踪→接近→攻击→逃跑/反击 | 基础 PvE |
| **领地战斗** | 入侵→警告→驱赶→短暂冲突→一方撤退 | 资源争夺 |
| **群体战斗** | 分工→站位→包围→诱饵→撤退→保护 | 小队战术 |
| **环境战斗** | 引诱→利用地形（酸池/毒气/窄道/高台） | 策略玩法 |
| **部件破坏** | 命中特定部件→功能下降（腿伤→减速，眼伤→失明） | 深度战斗 |

## 6.2 攻击类型

| 攻击 | 判定方式 | 效果 |
|------|----------|------|
| Bite 咬击 | Sphere overlap + 朝向角度 | 高单体伤害，出血，小目标可拖拽 |
| Charge 冲撞 | Capsule/AABB sweep | 击退，打断，撞入危险地形 |
| Slash 爪击 | 前方 OBB / 扇形 Sphere | 中距离近战，打断轻型单位 |
| Tail Swipe 尾击 | Sphere chain 轨迹 | 范围击退，防包围 |
| Projectile 骨刺 | Sphere 飞行体 | 远程伤害，中毒，减速 |
| Grapple 抓取 | Cone/Sphere + 体重上限 | 拖拽，持续消耗体力 |

## 6.3 克制关系

```
高视觉 → 克制伪装    高嗅觉 → 克制黑暗    重甲 → 克制小型爪牙
酸液 → 克制重甲      速度 → 克制慢速炮台    群体 → 克制单体强敌
范围攻击 → 克制群体   恐惧抗性 → 克制精神攻击
```

## 6.4 伤害结算

攻击命中 → Attack - Defense（最低伤害 1）→ HP 变化 → 可能触发 Buff（出血/中毒/减速）→ HP ≤ 0 → Dead → 尸体

# 7. 地图与生态

## 7.1 地图结构

地图 = 语义区域容器，不是装饰。

```
Room（生态小容器）
├── Habitat Zones（光照/湿度/温度/地形标签）
├── Resource Nodes（资源点）
├── Hazard Volumes（危险区）
├── Den Points（巢穴）
├── Sight Blocks（视野阻挡）
└── Portals（房间连接，限制体型/能力）
```

## 7.2 生态地图类型

| 地图 | 结构 | 核心测试 | 适合物种 |
|------|------|----------|----------|
| 开阔盆地 | 大平地，少量遮挡 | 速度，追击，逃跑路线 | 高速捕食者，群体猎手 |
| 窄洞网络 | 狭窄通道，小房间 | 感知，伏击，体型限制 | 小型毒虫，清道夫，蛇形 |
| 垂直脚手架 | 高低平台，跳跃点 | 高度差，跳跃/飞行能力 | 飞行捕食者，攀爬采集者 |
| 危险湿地区 | 酸池，浮岛，毒气 | 抗性，路线规划 | 酸抗生物，两栖生物 |
| 巢穴区 | 中心巢穴，多入口 | 潜入，偷资源，群体战 | 群居虫群，守卫型 |

## 7.3 物种分布

不是随机刷怪，而是算适应度：

```
SpawnScore = TerrainMatch + FoodAvailability + ShelterAvailability
           + TemperatureMatch + MoistureMatch
           - PredatorPressure - Competition
```

## 7.4 生态变量

地图持续模拟：食物量、尸体量、污染值、恐慌值、捕食压力、温度、光照、噪音。

```
尸体量高 → 清道夫增加    污染值高 → 突变体出现
恐慌值高 → 小动物减少    食物短缺 → 捕食者攻击性提高
```

## 7.5 生物生态位（8 类）

| 生态位 | 特征 | 作用 |
|--------|------|------|
| 小型猎物 | 快/血少/繁殖快 | 食物链底层 |
| 清道夫 | 吃尸体/嗅觉强 | 清理战场 |
| 伏击捕食者 | 低移动/高爆发/暗处 | 惩罚盲目探索 |
| 追击捕食者 | 快/耐力高/开阔地强 | 制造逃跑压力 |
| 重甲守卫 | 慢/防御高/守资源 | 构筑检验 |
| 群体虫群 | 单体弱/数量多/共享警报 | 测试 AoE/通道 |
| 环境生物 | 植物/菌毯/陷阱 | 生态容器 |
| Apex 生物 | 强/稀少/有领地 | 长期目标 |

# 8. 任务类型

| 任务 | 目标 | 测试能力 |
|------|------|----------|
| 回收 | 进入危险区→找到资源→带回 | 寻路、生存、携带、撤退 |
| 生态观察 | 观察野生生物→记录行为 | 潜行、感知、距离控制 |
| 狩猎 | 击杀或捕获目标 | 攻击策略、追踪、包围 |
| 护送 | 保护脆弱单位穿过危险区 | 队伍 AI、保护规则、队形 |
| 污染清理 | 清除污染源 | 环境抗性、续航 |
| 斗兽场 | 用当前生物击败敌方设计 | 构筑强度、战斗 AI |
| 适应性试炼 | 特定条件下存活/完成任务 | 系统理解深度 |

# 9. 进度系统

## 9.1 三种资源

| 资源 | 用途 | 来源 |
|------|------|------|
| **Biomass 生物质** | 孵化生物、修复身体、肉体器官 | 尸体、植物、狩猎 |
| **Alloy 合金** | 机械部件、传感器、装甲、基地升级 | 废墟、机器人、矿物 |
| **Data 数据** | 解锁 AI 模块、感知规则、蓝图复杂度 | 观察行为、扫描、**失败分析** |

> **失败也能给 Data** — 这是关键设计。

## 9.2 科技树

5 条线：Body 身体工程 / Sensor 感知技术 / Brain AI脑模块 / Habitat 基地设施 / Field Tools 现场工具

## 9.3 实验站升级

不升级玩家角色，升级实验站：孵化槽、脑容量上限、可部署数量、样本储存、训练场、模拟器、基因库、部件工坊。

# 10. 反馈与可视化

## 10.1 任务中反馈

头顶意图图标、当前状态颜色、目标连线、感知范围可视化、脚印/气味/声音波纹、恐惧/饥饿/愤怒条、攻击目标高亮、撤退路径虚线。

## 10.2 脑内视图（AI 调试 UI）

```
当前状态：Combat
最高意图：Flee 82
第二意图：Attack 61
触发规则：
  +50 血量低
  +30 敌人数量多
  +20 没有队友
  -18 当前攻击惯性
```

**这不是开发调试，是玩法 UI。** 没有它，玩家看不懂 AI 为什么这么做。

## 10.3 战后报告

成功/失败、耗时、资源收益、死亡原因、触发最多的规则、未执行目标的原因、AI 改进建议、环境适应性评分。

## 10.4 回放系统

任务录像、时间线、关键决策节点、死亡前 15 秒、规则触发记录、生态事件记录。可暂停、回看、单步查看。

# 11. 游戏模式

| 模式 | 特点 | 风险 |
|------|------|------|
| **实验场** | 安全测试，无永久损失，资源收益低 | — |
| **野外任务** | 主要玩法，有目标/风险/收益 | 生物可死亡 |
| **生态沙盒** | 投放生物观察长期生态变化 | — |
| **斗兽场** | 构筑验证，1v1/3v3/群体战 | — |
| **远征 Roguelike** | 连续地图，资源有限，死亡不可完全回滚 | 最高风险最高收益 |

# 12. 美术方案

适合 AI 辅助美术：怪物允许不完美、低模掩盖细节、部件化可复用。

路线：AI 概念图 → Blender 统一风格 → 低模化 → 统一材质 → 变成 BodyPart

弱化人类角色，视觉重点：怪物、机械生物、低模生态、环境模块、实验设施。

---

# 13. 关键风险

| 风险 | 解决 |
|------|------|
| 玩家看不懂 AI | 强制做脑内视图 + 决策解释 + 战后日志 |
| 不能直接控制很烦 | 有限但强的间接指挥 + AI 不听话有合理原因 |
| 构筑太复杂 | 预设模板 + 卡片规则 + 推荐修改项 |
| 所有玩家找最优解 | 环境多样化 + 部件维护成本 + 生态反制 |
| 美术风格不统一 | 统一低模风格 + 统一材质色板 + 统一部件接口 |
