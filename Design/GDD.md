# 1. 玩家身份

拾荒者 / 陪伴者。不是指挥官，是同伴。你和你创造的生命一起探险未知世界。

---

# 2. 核心循环

```
获得零件，组装生物（定义能力和性格）
↓
释放进入未知环境（战争迷雾，传感器是唯一信息来源）
↓
生物自主行动（探索/犹豫/战斗/逃跑/拾取）
↓
你通过它的感知"看到"世界
↓
关键节点喊话干预
↓
结果 → 理解它 → 强化它 → 进入更深的未知
```

## 决策密度

不是用 clicks/min 衡量。是用 **information updates / 10s** 衡量。

每 5-10 秒玩家必须从生物那里得到一条新信息（传感器揭示、停下、犹豫、发现、遭遇）。

---

# 3. 感知与未知

详见 [Sensory_Design.md](Sensory_Design.md)。

核心要点：
- 三层信息：真相 / 生物认知 / 玩家认知
- 视觉 = 实景揭开（MVP）
- 嗅觉 = 气流方向，无精确位置（MVP）
- 听觉 = 声波涟漪，无方位精度（Post-MVP）
- 每种感知产生不同的 AI 反应
- 地图 = 信息节拍编排

---

# 4. 生物性格系统

## 4.1 性格由零件决定 `MVP`

装什么零件就产生什么性格。不需要单独设定。

## 4.2 AI 模块可矛盾叠加 `MVP`

```
只装勇气核心：看到敌人 → 冲
勇气 + 谨慎：高血量冲，低血量逃 → 中间血量犹豫
贪欲 + 谨慎：想拿资源但害怕 → 纠结
```

**矛盾叠加产生的犹豫是张力核心。**

## 4.3 短期记忆 `MVP`

生物记住最近的遭遇：

```
被攻击过的方向 → 下次更警惕
发现过资源的位置 → 下次优先去
被你救过的经历 → 下次更听你的信号（信任度）
```

**没有历史敏感，"理解我的生物"就是假的。**

## 4.4 信任度 `Post-MVP`

你和生物之间的关系是双向的：

```
你发信号救了它 → 信任度 +20 → 下次犹豫时更听你的
你发信号坑了它（让它冲结果死了）→ 信任度 -30 → 下次不太听了
信任度低到一定程度 → 它完全忽略你的信号
```

**这是自走棋永远做不出来的深度 — 棋子没有记忆没有关系。**

# 5. 小队系统

## 5.1 一次投放 1-2 只生物 `MVP`

MVP 先做 1 只。M6 验证 2 只是否成立（注意力分配是否焦虑）。

**如果 2 只导致焦虑，回到 1 只路线。** 不硬撑。

## 5.2 多生物的意义 `Post-MVP`

不是"前后排配合打输出"。而是：

```
一只走左边，一只走右边
你同时关注两只的传感器
左边发现敌人，右边发现宝箱
你要决定：给左边发信号撤退？还是给右边发信号快拿？
```

**注意力分配本身就是游戏。** 但必须验证人类能承受。

# 6. 干预系统

## 6.1 干预是"喊话"，不是"操控" `MVP`

你是同伴，不是指挥官。喊话动词有人际感：

| 动词 | 效果 | 性格敏感 |
|------|------|----------|
| **警告** | AvoidThreat +30 | 勇气核心可能忽略 |
| **建议（去那边）** | MoveToTarget +30 | 谨慎核心可能不听 |
| **安抚** | Caution +20, Aggression -20 | 勇气核心听不进 |
| **鼓舞** | Aggression +20, Caution -20 | 谨慎核心听不进 |

**每个动词在不同性格身上响应不同。** 这才有"沟通的不完美"感。

## 6.2 干预资源 `MVP`

干预是稀缺的 — 探索中获得信号道具，一次性使用。

# 7. 零件系统

## 7.1 零件是获得的，不是选择的 `MVP`

探索中随机获得。背包 8 格。满时必须丢弃。

## 7.2 零件类型 `MVP`

| 类型 | 示例 |
|------|------|
| 身体部件 | 快腿、厚甲、毒牙、嗅觉腔 |
| AI 模块 | 勇气核心、谨慎核心、贪欲核心 |
| 信号道具 | 警告标记、侦察探针、鼓励脉冲 |

## 7.3 核心零件 `MVP`

Run 开始选的起始特征是核心零件。死亡不丢。

## 7.4 Build Identity `MVP`

生物在 Run 内持续存在。零件装上就留下。玩家能说出"我这只生物是侦察型"。

---

# 8. 地图与环境

## 8.1 小而密的迷雾房间 `MVP`

不是大而空的地图。是小而密、每 5-10 秒有信息事件的房间。

## 8.2 环境变量 `MVP`

| 变量 | 影响 |
|------|------|
| 光照 | 视觉范围变化（亮区大/暗区小） |
| 噪音 | 听觉检测范围变化（安静区灵敏/嘈杂区失灵） |
| 气味浓度 | 嗅觉检测范围变化 |

## 8.3 战争迷雾 `MVP`

生物走到哪里揭开哪里。已揭开区域保持可见。

---

# 9. 战斗

## 战斗是"事件"，不是"高潮" `MVP`

**反直觉的设计**：战斗短而突然，3-5 秒接触，结果立刻可解释。

```
生物遇到敌人
→ 3 秒接触（前摇 0.3 秒 + 扣血 + 音效 + 伤害数字）
→ 一方死亡或撤退
→ 继续探索
```

战斗的紧张感不来自打击感，来自：
- "它会撑住吗？"
- "它会撤吗？"
- "我要不要喊它撤？"

**这些都是性格读取，不是动作读取。**

# 10. 反馈

## 10.1 感知可视化

详见 [Sensory_Design.md](Sensory_Design.md)。

## 10.2 性格高亮 `MVP`

做了什么 + 没做什么：

```
它选择了进攻（勇气核心主导）
它没拿那个资源（勇气核心让它优先打敌人）
它犹豫了（勇气想打，谨慎想跑）
```

## 10.3 犹豫时刻 `MVP`

"？" + 减速 + Brain View 浮现。

## 10.4 结果 `MVP`

任务成功/失败 + 获得/失去零件 + 性格标签。

# 11. 关键风险

| 风险 | 解决 |
|------|------|
| 探索节奏太慢 | 地图小而密，信息事件每 5-10 秒 |
| 看不懂生物在想什么 | 感知可视化 + 性格高亮 + 未选择高亮 |
| 战斗抢走注意力 | 短而突然，定位是事件不是高潮 |
| 多生物焦虑 | M6 验证，不成立就回到单生物 |
| 性格可预测就无聊 | 短期记忆 + 环境敏感 + 矛盾叠加 + 信任度 |
