# 1. 玩家身份

**荒野生物工程师** — 边境实验站的管理员。

玩家在废土上建立实验站，通过回收基因、骨骼、机械部件，创造适应环境的生物去探索危险生态区。

**玩家不在世界里。** 你只能通过你创造的生物去"看"这个世界。

---

# 2. 核心循环

## 2.1 Run结构

每次Run是一次完整的探险：

```
你在实验站组装1-3只生物
↓
出发 → 第1个生态区（简单）
↓
收集资源/部件，遇到敌人
↓
继续 → 第2个生态区（更难）
↓
继续 → 第3个生态区（更难）
↓
...直到你撤退或全灭
↓
Run结束：带回所有收集的资源/部件
↓
meta进度：解锁新部件选项、新AI规则、新生态区
↓
下次Run带更好的设计出发
```

**核心问题：这次能走多远？**

## 2.2 Run内资源 vs Meta资源

| 资源类型 | 来源 | 死亡后 |
|---------|------|--------|
| **Run内部件** | 生态区内拾取/掉落 | **丢失** |
| **Run内信号** | 生态区内拾取 | **丢失** |
| **Run内食物** | 生态区内拾取 | **丢失** |
| **样本** | 完成任务目标 | **保留**（即使死亡） |
| **数据** | 观察行为/失败分析 | **保留** |
| **解锁** | 用数据/样本解锁 | **永久** |

**关键设计**：
- Run内资源是"这次能用的"，死了就没了
- 样本和数据是"meta进度"，死了也保留
- 解锁是"永久扩展"，下次Run有更多选项

## 2.3 撤退机制

```
任何时候你可以选择撤退。
撤退 = 带回当前所有收集的资源/部件/样本。
但放弃后面生态区的奖励。
```

**撤退是策略核心**：
- 继续深入 = 高风险高回报
- 撤退 = 安全但放弃机会
- 全灭 = 丢失所有Run内资源，只保留样本和数据

## 2.4 玩家核心动词（按时间占比）

| 动词 | 时间占比 | 操作 |
|------|---------|------|
| **设计** | 20-30% | 选部件、配AI蓝图、调参数（Run前+Run内） |
| **观察** | 30-40% | 读生物状态、读环境、读反馈 |
| **决策** | 20-30% | 继续还是撤退？用哪个信号？赌不赌？ |
| **分析** | 10-15% | 战后报告、改造方向 |

# 3. 身体部件系统

## 3.1 部件类型

生物由8类部件组成：

### Core 核心

| 类型 | 特点 |
|------|------|
| 轻型核心 | 便宜、轻、脆 |
| 重型核心 | 血厚、慢、耗能高 |
| 群体核心 | 适合多单位协作 |
| 捕食核心 | 攻击欲和追踪能力强 |
| 实验核心 | 可装更多AI模块，但不稳定 |

### Locomotion 移动器官

| 类型 | 特点 |
|------|------|
| 双足 | 均衡 |
| 四足 | 稳定，适合追猎 |
| 六足 | 地形适应强 |
| 蛇形 | 可钻缝 |
| 翅膀 | 跨越地形，能耗高 |

### Sensor 感知器官

| 类型 | 特点 |
|------|------|
| 普通眼 | 基础视觉 |
| 夜视眼 | 黑暗可见 |
| 热感器 | 发现隐藏生物 |
| 大耳 | 听觉范围大 |
| 嗅觉腔 | 追踪血迹/尸体/巢穴 |
| 危险腺体 | 感知毒气、火、酸 |
| 群体天线 | 共享同伴情报 |

### Weapon 攻击器官

爪、牙、角、毒刺、喷酸腺、电击器、远程骨刺、机械锯

### Defense 防御器官

甲壳、鳞片、脂肪层、再生组织、反伤尖刺、热抗皮肤、酸抗脚垫、伪装皮肤

### Utility 功能器官

携带囊、挖掘爪、采集口器、开门工具、能量电池、隐身膜、跳跃囊、攀爬钩

### Brain 脑模块

| 类型 | 特点 |
|------|------|
| 原始脑 | 只能执行少量规则 |
| 猎手脑 | 攻击、追踪、伏击加成 |
| 群体脑 | 队友协作规则更多 |
| 工蜂脑 | 采集、搬运、建造加成 |
| 战术脑 | 可使用掩体、包抄、撤退 |

### Social 社交/群体器官

信息素腺体、吼叫器官、信号天线、群体同步神经、恐吓面具

## 3.2 部件属性

每个部件带标签：

```
重量、能耗、噪音、耐久、维修难度
适配地形、兼容部件、AI需求
```

## 3.3 预算系统

```
体重预算、能量预算、脑容量预算
```

**构筑有取舍。** 不能堆满最强部件。

## 3.4 Run内部件获取

```
生态区内拾取（宝箱/资源点）
敌人掉落
任务奖励
```

**Run内部件装上就留在生物身上，直到Run结束或生物死亡。**

## 3.5 Meta解锁

用数据和样本解锁新部件选项：

```
数据x100 → 解锁"夜视眼"选项
样本x5 → 解锁"酸性洞穴"生态区
```

**解锁 = 增加下次Run的起始选项。** 不是直接变强。

# 4. AI蓝图系统

## 4.1 卡片式规则

```
当【看到敌人】
并且【敌人距离 < 5m】
并且【自己血量 > 50%】
则【近战攻击倾向 +60】
```

玩家组合：条件卡、意图卡、权重。

**编译目标**：卡片规则最终编译成 [System_Core §6](System_Core.md#6-效用与犹豫决策且可归因) 的 `PersonalityBias[intent]` 和 `Relevance(intent, belief.Tag)`。

## 4.2 AI蓝图结构

```csharp
public sealed class AiBlueprint
{
    public string BlueprintId;
    public List<AiRule> Rules;
    public Dictionary<string, float> BlackboardVars;
    public StateMachine States;
}
```

## 4.3 基础状态

```
Idle / Explore / Forage / Combat / Flee / Recover / Return / Protect / Nest
```

## 4.4 意图 Intent

```
MoveToTarget / AttackTarget / FleeFromThreat / EatFood
InvestigateStimulus / ReturnToNest / ProtectAlly / CarryItem
UseAbility / Hide / CallForHelp / Rest
```

## 4.5 内部需求

行为 = f(AI蓝图, 需求, 环境, 记忆)，不只是f(AI蓝图)。

需求通过 `NeedWeight[need]` 接入 [System_Core §6](System_Core.md#6-效用与犹豫决策且可归因) 的效用公式。

| 需求 | 范围 | 效用公式中的作用 |
|------|------|----------------|
| 饥饿 | 0-100 | NeedWeight[Hunger] 放大进食 Intent |
| 疲劳 | 0-100 | NeedWeight[Fatigue] 放大休息 Intent |
| 恐惧 | 0-100 | NeedWeight[Fear] 放大逃跑 Intent |
| 好奇 | 0-100 | NeedWeight[Curiosity] 放大调查 Intent |
| 攻击欲 | 0-100 | NeedWeight[Aggression] 放大攻击 Intent |
| 思乡 | 0-100 | NeedWeight[Homesick] 放大返回 Intent |

# 5. 玩家介入工具

## 5.1 信号工具

| 信号 | 效果 |
|------|------|
| 移动信标 | 提高前往区域倾向 |
| 危险标记 | 提高避开区域倾向 |
| 召回信标 | 提高返回倾向 |
| 安抚波 | 降低恐惧/愤怒 |
| 刺激波 | 提高攻击/速度，增加失控风险 |

**信号是Run内资源。** 用完就没了。必须选择什么时候用。

## 5.2 环境工具

投放诱饵、投放食物、制造噪音、释放气味、点火、关灯

## 5.3 紧急工具

紧急传送（救回生物但部件损坏）、镇静剂、自毁命令

# 6. 任务与生态区

## 6.1 Run结构

每次Run = 3-5个连续生态区，难度递增：

| 位置 | 生态区 | 难度 |
|------|--------|------|
| 第1区 | 废弃温室 | T1 |
| 第2区 | 酸性洞穴 | T2 |
| 第3区 | 机械废墟 | T2-T3 |
| 第4区 | 火山地带 | T3 |
| 第5区 | 异常区 | T3+ |

**每个区的房间/敌人/资源是随机的。**

## 6.2 生态区内容

每个生态区包含：
- 3-5个房间（随机拼接）
- 野生生物（有自己的生态位）
- 资源/部件
- 任务目标（收集样本/击杀目标/观察生物）
- 出口（通向下一个区）

## 6.3 撤退点

每个生态区出口前有撤退选项：
- 选择"继续" → 进入下一个区
- 选择"撤退" → Run结束，带回所有收集的资源

**撤退是策略核心。** 继续深入风险更高但奖励更好。

## 6.4 任务类型

| 类型 | 目标 |
|------|------|
| 回收任务 | 找到资源/样本并带到出口 |
| 狩猎任务 | 击杀目标生物 |
| 观察任务 | 记录野生生物行为数据 |
| 护送任务 | 保护脆弱单位穿过危险区 |

# 7. 感知与未知

详见 [Sensory_Design.md](Sensory_Design.md)。

核心要点：
- 三层信息：真相 / 生物认知 / 玩家认知

# 9. 战后报告

任务结束必须有完整报告：

```
Run结果：第3区撤退
收集：样本x5，部件x3（酸抗脚垫/夜视眼/毒刺）
生物状态：钉子HP45%，锤子死亡（第2区）

关键决策：
- 第2区：锤子冲了机械巡逻者，死亡
- 第3区：钉子HP下降，选择撤退

问题：
- 锤子AI没有"观察强敌"规则
- 钉子缺少热抗性（第3区高温）

建议：
- 锤子添加规则：看到强敌→观察+80
- 下次Run解锁热抗皮肤
- 考虑带3只生物分散风险

解锁进度：
- 数据x50（用于解锁夜视眼）
- 样本x5（用于解锁酸性洞穴）
```

**战后报告是L4认知循环的输入端。**

# 10. Meta进度

## 10.1 实验站设施

| 设施 | 功能 |
|------|------|
| 孵化槽 | 创造生物（槽位数量随升级增加） |
| 部件仓库 | 存储解锁的部件选项 |
| AI蓝图台 | 配置行为规则 |
| 实验场 | 安全测试生物 |
| 分析屏 | 查看战后报告 |

## 10.2 成长资源

| 资源 | 用途 | 来源 |
|------|------|------|
| **样本** | 解锁新生态区、新部件选项 | 完成任务目标（死亡也保留） |
| **数据** | 解锁新AI规则、研究野生生物 | 观察行为/失败分析（死亡也保留） |

## 10.3 解锁不是变强，是扩展选项

```
解锁"夜视眼" → 下次Run可以选择装它（不是自动装上）
解锁"酸性洞穴" → 下次Run可以选择去那里（不是必须去）
解锁"观察强敌规则" → 下次Run可以配置到AI蓝图里
```

**每次Run的起始状态是固定的。** 解锁只是增加选择，不直接变强。

# 11. 战斗

战斗是"事件"，不是"高潮"：

```
生物遇到敌人
→ 0.3秒前摇
→ 3秒接触
→ 一方死亡或撤退
→ 继续探索
```

# 12. 关键风险

| 风险 | 解决 |
|------|------|
| 玩家看不懂AI在干什么 | 强反馈：意图UI + 决策解释 + 战后日志 |
| 不能直接控制很烦 | 有限但强力的间接指挥 |
| 构筑太复杂新手进不来 | 预设模板 + 卡片规则 + 失败后建议 |
| 所有玩家找到最优解 | 环境多样化 + 生态反制 |
| Run太短没深度 | 3-5个区，每区5-12分钟 |
| Run太长太累 | 撤退机制，可以随时退出 |
