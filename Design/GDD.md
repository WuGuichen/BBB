# GDD — Creature Roguelike 玩法设计

> 版本 0.6 | 2026-05-27
>
> 每个系统标注阶段：`MVP` / `Post-MVP` / `Long-term`

---

# 1. 玩家身份

拾荒者 / 探险者。在废弃实验设施和危险生态区中探索，收集零件，组装造物，完成任务，深入更危险区域。

不是实验室管理员。是**在野外捡到什么就用什么的人**。

---

# 2. 核心循环

## 一次 Run `MVP`

```
选择进入一个区域
↓
探索（地图逐步揭开，发现敌人/资源/零件/出口）
↓
获得随机零件（2-3个，从区域零件池抽取）
↓
用当前手牌组装生物（部件 + AI 模块）
↓
投放生物执行任务目标
↓
生物自动行动（探索/战斗/拾取/逃跑）
↓
关键时刻用信号干预（每次 Run 有限次数）
↓
成功 → 保留零件，进入下一区域
失败 → 失去部分零件，尝试下一区域或重来
```

## 单次投放 `MVP`

一次投放 30-60 秒。不长。

```
投放
↓
生物自主探索（5-10秒）
↓
遇到第一个选择点（打/绕/拿）（张力时刻）
↓
继续行动（10-15秒）
↓
遇到第二个选择点（可能的信号干预点）
↓
完成或失败（30-60秒总时长）
```

**任务短，但每个任务都有至少一个"关键时刻"。**

## 决策密度

一次 Run（3-5 个区域，每个 30-60 秒投放）：
- 每个区域：1-2 个组装决策（拿到新零件后怎么调整）
- 每次投放：1-2 个信号决策
- 每个 Run：3-5 个"这个零件值不值得冒险拿"的决策

# 3. 零件系统

## 3.1 零件不是选择，是获得 `MVP`

玩家不能在实验室里自由配置。零件来自探索：

- 击败敌人掉落
- 宝箱/隐藏区域
- 任务奖励
- 商店（Post-MVP）

**掉什么随机。** 从当前区域的零件池里抽取。

## 3.2 零件类型 `MVP`

| 类型 | 作用 | 示例 |
|------|------|------|
| **身体部件** | 改变生物的物理能力 | 快腿、厚甲、毒牙、嗅觉腔、翅膀 |
| **AI 模块** | 改变生物的行为倾向 | 勇气核心（攻击+）、谨慎核心（逃跑+）、贪欲核心（拾取+） |
| **信号道具** | 一次性干预工具 | 召回信标、危险标记、鼓励脉冲 |

## 3.3 物品栏限制 `MVP`

玩家有一个有限的零件背包。

```
背包容量：8 格
身体部件占 1-2 格（大件占 2 格）
AI 模块占 1 格
信号道具占 1 格，可叠放（最多 3 个同类）
```

**背包满时必须丢弃才能拿新的。** 这本身就是策略决策。

## 3.4 组装 `MVP`

每次投放前，玩家组装一只生物：

```
身体：选择 1-2 个身体部件
AI：选择 1-2 个 AI 模块
信号：选择携带哪些信号道具（从背包装入信号槽，最多 2 个）
```

组装时间：10-15 秒。快速决策，不慢慢配。

## 3.5 零件稀有度 `MVP`

| 稀有度 | 效果 | 掉率 |
|--------|------|------|
| 白色 | 基础效果 | 高 |
| 蓝色 | 更强效果 或 带副作用 | 中 |
| 紫色 | 强效果 + 独特交互 | 低 |

示例：

```
白色快腿：移速 +20%
蓝色快腿：移速 +35%，但噪音 +50%
紫色快腿：移速 +25%，穿越敌人时获得短暂隐身
```

# 4. 生物性格系统

## 4.1 性格不是旋钮，是零件决定的 `MVP`

生物没有单独的"性格设定"。性格由装了什么零件自然产生。

```
装了勇气核心 + 毒牙 → 这是一只好斗的猎手
装了谨慎核心 + 快腿 + 嗅觉腔 → 这是一只警觉的侦察者
装了贪欲核心 + 大背包 → 这是一只冒险的拾荒者
```

**玩家不需要"设定性格"，装零件就是在定义性格。**

## 4.2 性格要鲜明 `MVP`

每种 AI 模块的效果必须大到能明显改变行为：

```
勇气核心：看到敌人时攻击倾向 +60（大幅）
谨慎核心：看到敌人时逃跑倾向 +60（大幅）
贪欲核心：看到资源时拾取倾向 +60（大幅）
急行核心：移动速度 +40%，但停下来犹豫的时间缩短
回巢核心：离开起点超过一定距离后自动返回倾向 +50
```

**不是微调 0-100，是明确的"这只生物是什么性格"。** 一句话能描述。

## 4.3 性格可预测但不完全可控 `MVP`

玩家装了勇气核心，就知道它会主动打怪。但：

```
它可能追太深（勇气核心的代价）
它可能忽略资源（因为攻击优先级太高）
它可能在敌人太强时仍然冲（因为勇气核心压制了逃跑倾向）
```

**这就是乐趣来源 — "我就知道它会这样！"**

## 4.4 零件之间有交互 `Post-MVP`

```
勇气核心 + 毒牙 → 中毒的敌人会逃跑，制造追击机会
谨慎核心 + 嗅觉腔 → 提前发现威胁，更聪明地绕路
贪欲核心 + 快腿 → 冲到危险区域快速拾取再跑回来
```

这些交互不需要玩家背诵，而是在游戏中自然发现。

# 5. 地图与探索

## 5.1 战争迷雾 `MVP`

地图开始是黑的。生物走到哪里，哪里揭开。玩家看到的是生物的视角，不是上帝视角。

**这创造了信息稀缺 — 我不知道前方有什么，直到生物走过去。**

## 5.2 地图内容 `MVP`

每张地图包含：

| 内容 | 作用 |
|------|------|
| 敌人 | 威胁，也可能掉落零件 |
| 资源点 | 任务目标 |
| 零件宝箱 | 获得新零件（可能在危险位置） |
| 出口 | 完成任务后离开 |
| 危险地形 | 酸池/窄道/黑暗区 |

## 5.3 歧义设计 `MVP`

每张地图至少有 2 条路线，各有取舍：

```
安全路线：远，没有额外零件
危险路线：近，有零件宝箱但有敌人
```

**"值不值得冒险拿那个零件"是每局的核心策略问题。**

# 6. 战斗

## 6.1 自动战斗 `MVP`

生物遇到敌人后自动战斗。战斗结果由零件配置决定。

```
勇气核心 + 毒牙 + 厚甲 → 正面硬刚，大概率赢
谨慎核心 + 快腿 → 看到敌人就跑，不会死但拿不到掉落
贪欲核心 → 可能边打边捡，分散注意力
```

## 6.2 战斗过程 `MVP`

30 秒内的简短战斗：

```
生物发现敌人
→ 根据 AI 模块决定打/跑/绕
→ 接触伤害（碰到就受伤）
→ 持续几秒
→ 一方死亡或逃跑
```

MVP 不做复杂攻击判定（Bite/Charge）。只做接触伤害 + 生死。

## 6.3 战斗张力 `MVP`

战斗的张力不在操控，在观察：

```
它冲上去了 — 能赢吗？
它被打了一下 — 还撑得住吗？
它残血了 — 会跑吗？（取决于它的性格）
它跑了 — 松了一口气
它没跑 — 糟了
```

**性格在这里体现得最明显。** 勇气核心的生物残血了可能还在打。谨慎核心的生物半血就跑了。

# 7. 信号系统

## 7.1 信号是道具，不是能力 `MVP`

信号不是"每局 3 次的 cooldown"。信号是**你找到的道具**。

```
召回信标：让生物倾向返回出口（一次性）
危险标记：让生物倾向避开某区域（一次性）
鼓励脉冲：让生物暂时更勇敢（一次性）
```

信号在探索中获得。可能一局找到 3 个，可能一个都没找到。

**没有信号 = 没有干预能力 = 纯靠设计。** 这让信号变得珍贵。

## 7.2 信号使用 `MVP`

投放生物后，玩家可以在关键时刻使用信号。信号改变 AI 的 Utility 分数。

```
生物犹豫了（头顶"？"）
→ 我使用召回信标
→ ReturnToExit +80
→ 它回来了
```

**信号的价值在于"在正确的时间用在正确的地方"。** 这是操作层面唯一的技巧空间。

# 8. 进度系统

## 8.1 Run 结构 `MVP`

一次 Run = 3-5 个区域，难度递增。

```
区域 1：简单敌人，少量零件
区域 2：更多敌人，更好的零件
区域 3：强敌人 + 危险地形，稀有零件
Boss 区域（Post-MVP）
```

## 8.2 死亡与保留 `MVP`

Run 内死亡：
- 失去本次 Run 获得的零件
- 保留解锁的永久内容

Run 间进度（Post-MVP）：
- 解锁新的零件池
- 解锁新的区域类型
- 解锁新的信号类型

## 8.3 成就感来源 `MVP`

不是"我操作好"。而是：

```
我用一堆看似不搭的零件，拼出了一只能赢的生物
→ 这是策略成就感
→ 类似 StS 用垃圾牌打赢 Boss 的爽感
```

# 9. 反馈系统

## 9.1 性格行为展示 `MVP`

生物行动时，关键性格决策用一句话高亮：

```
它选择了进攻（勇气核心）
它绕开了危险（谨慎核心的嗅觉腔检测到威胁）
它偏离路线去拿资源（贪欲核心）
它残血了但还在打（勇气核心压制了逃跑）
```

**不是 Utility 数字。是性格的一句话描述。**

## 9.2 关键时刻 `MVP`

生物犹豫时（最高分和第二分差距小），画面放慢 + 头顶"？"：

```
它在犹豫
打还是跑？
（信号使用提示）
```

这是整个游戏的张力核心。

## 9.3 结果展示 `MVP`

每次投放结束：

```
任务成功 / 失败
获得零件：蓝色毒牙
失去零件：白色快腿（死亡时丢失）
生物存活 / 死亡
性格标签：鲁莽冲锋型 / 谨慎求生型 / 贪婪冒险型
```

# 10. 美术 `MVP: 占位体`

占位几何体 + 颜色区分稀有度（白/蓝/紫）。但必须有：

- 发现敌人：警觉音
- 犹豫时刻：画面轻微放慢 + 音效
- 拾取零件：满足感音效
- 任务成功：明确的胜利反馈
- 死亡：明确的失败反馈 + 展示失去了什么

---

# 11. 关键风险

| 风险 | 解决 |
|------|------|
| 间接控制不如手操 | 手操做不了 — 零件随机，策略才是核心 |
| 策略有最优解 | 每局零件不同，只有当前手牌的最优打法 |
| 看 AI 走路无聊 | 任务短（30-60秒），关键时刻有张力，性格鲜明 |
| 认知负担高 | 零件少、性格一句话可描述、结果一句话可解释 |
| 零件池太小重复感强 | Post-MVP 扩池；MVP 靠零件交互创造多样性 |
| 战斗太简单没深度 | MVP 先验证核心循环；Post-MVP 加 Bite/Charge/环境战斗 |

# 12. 长线扩展 `Post-MVP` `Long-term`

| 扩展 | 阶段 |
|------|------|
| 更多零件池（每区域独特掉落） | `Post-MVP` |
| 零件合成/升级 | `Post-MVP` |
| Boss 关卡 | `Post-MVP` |
| Bite/Charge/Projectile 攻击类型 | `Post-MVP` |
| 环境战斗（酸池/毒气） | `Post-MVP` |
| 多生物投放（小队） | `Post-MVP` |
| 永久解锁（新零件池/新区域/新信号） | `Post-MVP` |
| 生态模拟/物种分布 | `Long-term` |
| 玩家分享蓝图/种子 | `Long-term` |
| Mod 支持 | `Long-term` |
