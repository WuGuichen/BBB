# GDD — Creature Roguelike 玩法设计

> 版本 0.7 | 2026-05-27

---

# 1. 玩家身份

拾荒者 / 探险者。在废弃实验设施和危险生态区中探索，收集零件，组装造物小队，完成任务，深入更危险区域。

---

# 2. 核心循环

## 一次 Run `MVP`

```
Run 开始：选择起始特征（决定本局方向）
↓
查看星图：选择下一个节点
↓
进入节点：探索 + 获得零件
↓
组装/强化小队
↓
投放小队执行任务（20-40秒自动执行）
↓
用信号/扫描/路径标记干预
↓
成功 → 保留零件，小队成长
失败 → 失去非核心零件
↓
选择下一个节点
```

## 决策密度目标

每分钟至少 4 次有意义决策。来源：
- 组装/强化（每节点 1-2 次）
- 操作干预（每次投放 2-3 次）
- 路径选择（每节点间 1 次）
- 零件取舍（获得时 1 次）

## 单次投放 `MVP`

一次投放 20-40 秒。小队 2-3 只生物同时行动。

```
投放
↓
小队各自探索（5-8秒）
↓
遇到第一个选择点（打/绕/拿）
↓
性格冲突产生张力（勇的冲、怂的躲）
↓
操作干预（信号/扫描/路径标记）
↓
战斗或完成（20-40秒总时长）
```

# 3. 小队系统

## 3.1 一次投放 2-3 只生物 `MVP`

不是单生物。是小队。

```
小队槽位：2 个（MVP），3 个（Post-MVP）
每只生物：1-2 个身体部件 + 1 个 AI 模块
```

**小队制立刻解决**：
- 决策密度（编排谁前谁后、谁打谁捡）
- 协同空间（勇的吸引火力、怂的偷资源）
- 注意力分配（看哪只？信号给谁？）
- 失败梯度（一只死了不等于全军覆没）

## 3.2 队形/分工 `MVP`

投放时选择小队排列：

```
前排：承受伤害，优先接触敌人
后排：跟随，优先拾取/逃跑
```

简单的前后排就够了。不需要复杂阵型。

## 3.3 小队协同示例

```
生物 A：勇气核心 + 毒牙 → 前排，吸引火力，打敌人
生物 B：贪欲核心 + 快腿 + 嗅觉腔 → 后排，绕路拿资源，闻到危险就跑

结果：A 打赢了敌人，B 顺手拿了宝箱。双赢。
```

```
生物 A：谨慎核心 + 快腿 → 前排，侦察
生物 B：勇气核心 + 厚甲 → 后排，等 A 确认安全后冲上

结果：A 先发现敌人，B 后手进攻。配合。
```

```
生物 A：勇气核心 → 冲上去了
生物 B：贪欲核心 → 去捡旁边的资源了
都没有掩护对方

结果：A 被围殴死了，B 被敌人追上也死了。冲突。
```

**性格冲突不是 bug，是游戏性。** 玩家要预判"这两只放一起会怎样"。

# 4. 零件系统

## 4.1 零件是探索中获得的 `MVP`

零件来自：击败敌人掉落、宝箱、事件奖励。掉什么随机。

## 4.2 零件类型 `MVP`

| 类型 | 作用 | 示例 |
|------|------|------|
| **身体部件** | 改变物理能力 | 快腿、厚甲、毒牙、嗅觉腔 |
| **AI 模块** | 改变行为倾向 | 勇气核心、谨慎核心、贪欲核心 |
| **信号道具** | 一次性干预 | 召回信标、危险标记、鼓励脉冲、侦察探针 |
| **起始特征** | Run 开始选择 | "天生勇猛"（初始勇气核心）、"拾荒本能"（初始嗅觉腔） |

## 4.3 背包限制 `MVP`

```
背包容量：8 格
身体部件占 1-2 格
AI 模块占 1 格
信号道具占 1 格
```

背包满时必须丢弃才能拿新的。

## 4.4 零件稀有度 `MVP`

| 稀有度 | 效果 | 掉率 |
|--------|------|------|
| 白色 | 基础效果 | 高 |
| 蓝色 | 更强 + 有代价 | 中 |
| 紫色 | 强 + 独特交互 | 低 |

**蓝色/紫色的副作用必须真的能咬人。** 蓝色快腿"噪音+50%"在 MVP 如果没有听觉敌人就没有意义。MVP 的副作用必须在当前系统内有 gameplay 后果。

## 4.5 Build Identity `MVP`

**生物在 Run 内持续存在，不是每次投放重组。**

```
Run 开始：选起始特征 → 决定方向
区域 1：获得零件 → 装上，生物开始"成型"
区域 2：获得更多零件 → 强化或调整
区域 3：生物已经是"我这局的构筑"
```

**玩家在第 3 个区域能说出"我这局是毒牙冲锋流"或"我这局是侦察偷取流"。**

零件是基因不是装备 — 装上就留下了。死亡时可能掉非核心零件，但核心零件保留。

## 4.6 核心零件 `MVP`

Run 开始时选的起始特征和初始 AI 模块是"核心零件"。死亡时不丢。

```
"我这局的核心是勇气核心 + 毒牙"
→ 即使在区域 2 死了
→ 区域 3 我还有勇气核心 + 毒牙
→ 但失去了区域 2 获得的蓝色快腿
```

**这让失败有代价但不挫败。** 参考 Hades 永远是 Zagreus。

## 4.7 零件升级 `Post-MVP`

稀有资源能把白色升蓝色。创造"培养"情感。

# 5. AI 性格系统

## 5.1 性格由零件决定 `MVP`

装了什么零件就产生什么性格。不需要单独设定。

## 5.2 AI 模块可矛盾叠加 `MVP`

**这是关键设计。** 不是"装了勇气就永远冲"。而是：

```
只装勇气核心：看到敌人 → 冲
勇气核心 + 谨慎核心：高血量时冲，低血量时逃
→ 中间血量出现真正的"犹豫时刻"
```

```
只装贪欲核心：看到资源 → 去拿
贪欲核心 + 回巢核心：拿了资源就想回家
→ 拿到东西后出现"走还是继续"的犹豫
```

**矛盾叠加产生的犹豫是游戏的张力核心。** 纯单一模块的生物永远不犹豫，游戏就变成"看预设行为表演"。

## 5.3 性格高亮 `MVP`

关键决策时一句话高亮：

```
它选择了进攻（勇气核心主导）
它犹豫了——勇气核心想打，谨慎核心想跑（血量中等）
它没拿那个资源——勇气核心让它优先打敌人
```

**"没做某事"的高亮比"做了某事"更能让玩家理解 AI。**

# 6. 操作维度

## 6.1 信号 `MVP`

信号是道具，探索中获得。使用时改变 AI Utility 分数。

| 信号 | 效果 | 一次性 |
|------|------|--------|
| 召回信标 | ReturnToExit +80 | ✓ |
| 危险标记 | AvoidZone +30 | ✓ |
| 鼓励脉冲 | Aggression 临时 +30 | ✓ |

## 6.2 侦察 `MVP`

侦察探针是道具。使用时揭开局部地图。

```
背包带 1 个侦察探针
投放后使用 → 揭开前方 10 米范围
→ 发现敌人位置 / 宝箱位置 / 危险区
→ 然后决定信号怎么用
```

**情报本身就是资源。** 带不带探针是背包取舍。

## 6.3 撤退指令 `MVP`

随时可让小队撤退。不需要道具，但有代价：

```
撤退后该节点视为失败
失去本次获得的非核心零件
但保住了小队不死
```

**撤退是"止损"决策，不是免费操作。**

## 6.4 路径标记 `Post-MVP`

在地图上画建议路线。AI 加权但不强制。

# 7. 地图与星图

## 7.1 分支星图 `MVP`

每层 5-7 个节点，玩家每步选 1 条路。

```
节点类型：
普通：简单敌人 + 基础零件
精英：强敌人 + 好零件
事件：随机效果（可能好可能坏）
商店：用零件换零件（Post-MVP）
休息：恢复 HP / 升级零件
```

**路径选择本身是策略。** 走精英路拿好零件但风险高，走普通路安全但零件差。

## 7.2 单节点地图 `MVP`

每个节点是一张小地图：

```
多条路线 + 宝箱 + 敌人 + 资源点 + 出口
战争迷雾（生物走到哪里揭开哪里）
```

# 8. 战斗

## 8.1 自动战斗 `MVP`

生物遇到敌人后自动战斗。结果由零件配置决定。

## 8.2 战斗可观察性 `MVP`

接触伤害没错，但必须有可读性：

```
攻击有 0.3 秒前摇（抬手/蓄力）+ 音效
伤害数字弹出（白字普通/黄字暴击/灰字未命中）
HP 条变化
死亡时有明确反馈
```

**不需要复杂物理。但需要"我在 1 秒内看懂了这场战斗"。**

## 8.3 战斗张力 `MVP`

```
A 冲上去了 — 能赢吗？
A 被打了一下 — 还撑得住吗？
A 残血了 — 会跑吗？（取决于谨慎核心是否压制勇气核心）
B 在旁边偷资源 — 被敌人发现了！
我要不要用召回信标把 A 拉回来？
```

# 9. 进度系统

## 9.1 Run 结构 `MVP`

```
星图 3 层，每层 5-7 节点
每层末尾有精英节点（可选）
难度递增
```

## 9.2 死亡与保留 `MVP`

- 核心零件（起始特征 + 初始 AI 模块）永不丢失
- 非核心零件死亡时丢失
- 小队生物死亡时标记为"重伤"，下节点可恢复或替换

## 9.3 Run 间永久进度 `Post-MVP`

解锁新零件池、新区域、新起始特征。

# 10. 反馈

## 10.1 性格高亮 `MVP`

```
它选择了进攻（勇气核心主导）
它没拿资源（勇气核心让它优先打敌人）
它犹豫了（勇气想打，谨慎想跑）
它残血了但还在打（勇气压制了逃跑）
```

## 10.2 犹豫时刻 `MVP`

AI 内部矛盾时（最高分和第二分差距小），画面轻微放慢 + 头顶"？" + 信号使用提示。

## 10.3 结果展示 `MVP`

```
任务成功/失败
获得/失去零件
小队存活状态
性格标签（鲁莽冲锋型 / 谨慎求生型 / 冒险拾取型）
```

# 11. 关键风险

| 风险 | 解决 |
|------|------|
| 决策密度太低 | 小队制 + 多操作维度 + 分支星图 |
| 单体无协同 | 2-3 只小队，性格互补/冲突 |
| 接触伤害没可观察性 | 前摇+音效+伤害数字 |
| 没有 Build Identity | 生物持续存在，核心零件不丢 |
| 性格可预测就无聊 | AI 模块可矛盾叠加产生犹豫 |
| 操作面太窄 | 信号+侦察+撤退，3 个操作维度 |
| 零件随机挫败 | 核心零件死亡不丢 |
