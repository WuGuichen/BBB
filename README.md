# WGame_B

一款 Roguelike 即时策略游戏。用探索获得的零件组装有性格的生物，让它们在危险环境中自动执行任务。

> **用手里的牌，打出最好的局。**

## 当前状态

**阶段**：概念期 → 预制作期

**当前目标**：Roguelike Auto-Strategy Slice v0

验证：
- 探索中获得随机零件
- 用手里的零件组装生物
- 生物自动执行任务
- 关键时刻有张力
- 成功/失败有成就感
- 想再试一次

## 核心公式

```
稀缺零件 + 环境压力 → 倒逼策略设计 → 自动执行验证 → 成就感
```
